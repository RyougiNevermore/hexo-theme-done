# Hexo theme done


## Installation
    git clone git@github.com:RyougiNevermore/hexo-theme-done.git hexo/themes/done
    npm install pinyin
    npm install hexo-generator-tag
    npm install hexo-featured-image
    hexo new page about
    hexo new page tag
    hexo new page search


## Usage
No config.
